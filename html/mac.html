Основные команды:

ls — показывает список файлов и папок в текущей директории

cd [папка] — переход в указанную папку

cd .. — переход на уровень выше

pwd — показать текущий путь (Print Working Directory)

mkdir [название] — создать новую папку

touch [файл] — создать новый пустой файл

rm [файл] — удалить файл

rm -r [папка] — удалить папку и всё её содержимое

mv [старое] [новое] — переместить или переименовать файл/папку
